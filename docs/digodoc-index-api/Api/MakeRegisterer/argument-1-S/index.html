Parameter MakeRegisterer.1-S

Module Services defines all the services with their arguments, results and errors JSON encodings.

val section_indexation : EzAPI.Doc.section
val sections : EzAPI.Doc.section list

All documentation sections. Every service should be associated to one of them in order to appear in API documentation.

val generate : (Data_types.response, exn, EzAPI.no_security) EzAPI.service0

Service that indexates and generates entries for digodoc DB. Entries are extracted from special meta files generated previously by digodoc under the directory that should be specified in Index.docs_dir. Returns Data_types.response that indicates either indexation was succesful or not.

val sources : (Data_types.response, exn, EzAPI.no_security) EzAPI.service0

Service that indexates all digodoc sources files. Sources should be generated by digodoc under the directory that should be specified in Index.sources_dir. Returns Data_types.response that indicates either indexation was succesful or not.
